============================================
Advance/NeuralMD
============================================

Neural Network Potentialに基づいた分子動力学計算のソフトウェアです。\ `Quantum ESPRESSO <http://www.quantum-espresso.org/>`_\ にて出力された第一原理計算の結果を教師データとして、分子力場を作成します。この力場を利用して、\ `LAMMPS <http://lammps.sandia.gov/>`_\ にて分子動力学計算を実行します。
